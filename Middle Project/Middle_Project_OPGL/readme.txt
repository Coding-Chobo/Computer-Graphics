=====================구현한 내용======================
화면 좌측 우측에서 일정한 시간초마다 도형이 생성되고
반대방향으로 날라간다(예 : 왼쪽에서 오른쪽, 오른쪽에서 왼쪽)

날라가는 경로 : 곡선을 그린다.

도형의 종류 : 정3각형부터 정 20각형까지 존재, 하트 모양도 존재

아래에는 고양이가 방향키를 통해 이동하며 잘린 도형과 충돌하면 해당 도형의 rgb평균값에서 일정값으로 나눈 값이 회색조로 추가가 된다.

마우스를 통해 클릭한 지점부터 놓는 지점까지의 선분으로 도형을 슬라이스한다. 이때 도형이 잘리면 수직낙하가 아닌 아주 잠깐 튀어올랐다 양방향 곡선으로 잘려나간다 잘린 도형의 중심이 원 도형의 바깥부분일수록 낙엽처럼 나풀거리며 떨어진다.

슬라이스하지 못한 도형들은 화면을 나갔을때 삭제한다.(경로를 활성화하여 볼 수 있다.)

키보드 명령어
- l : 도형의 모드전환 ( line과 fill모드로 )
- p : 도형의 이동경로 그리기 활성화
- +/- : 날라오는 속도 변경(빨라지기 / 느려지기)
- q : 프로그램 종료

=====================추가로 구현한 내용======================
도형이 날라갈때 회전하며 날라간다.
회전된 상태에서 그대로 잘린다.

도형이 날라가는 경로가 곡선으로 날라간다

도형이 생성될때 20각형까지 생성한다.
하트 모양으로도 생성한다.

고양이가 잘린 도형을 만났을때 색상을 변경해나간다.
-고양이가 잘린 도형과 충돌하면 해당 도형의 rgb평균값에서 일정값으로 나눈 값이 회색조로 추가가 된다.

슬라이스한 도형을 다시 슬라이스할 수 있다.(횟수 제한 X)

슬라이스한 도형이 떨어지는 방식에 여러 로직 추가
 이때 도형이 잘리면 수직낙하가 아닌 아주 잠깐 튀어올랐다가 양방향 곡선으로 잘려나간다.
 잘린 도형의 중심이 원 도형의 바깥부분일수록 낙엽처럼 나풀거리며 떨어진다.
